# 藥品標記建議 (Medicine Labeling Guide)

針對藥品與藥盒辨識，為了在執行 `onnxruntime-web` 的手機端獲得更好的準確率，請遵循以下規範：

## 1. 標記框 (Bounding Box) 規範
*   **完整框選**：如果藥盒在畫面中是完整的，請框住整個藥盒。
*   **局部標記**：如果藥品過於大型且具有識別性的 Logo（例如：特定品牌的標誌），除了標記整個藥盒外，亦可多標一個「Logo」類別。
*   **不留白**：邊框應儘量緊貼藥品邊緣。

## 2. 角度多樣性 (Perspective)
*   **多面拍攝**：
    *   正面 (Front) ── 最重要的資訊區。
    *   側面 (Side) ── 包含成分與製造商。
    *   頂面 (Top) ── 常用於俯視角度辨識。
*   **模擬手持**：藥盒常被拿在手上，因此請也上傳有「人手」在背景的圖片，避免模型誤將手掌偵測為物體。

## 3. 背景與環境 (Background)
*   **真實場景**：除了純色桌面，請標記一些放在亂糟糟桌面、藥櫃、甚至是病人床頭櫃的圖片。
*   **陰影處理**：在強光下藥盒會有明顯陰影，標記時請不要將陰影包含在框內。

## 4. 常見問題 (FAQ)
*   **Q: 兩盒藥疊在一起怎麼辦？**
    *   A: 兩盒都分別畫框。如果遮擋太嚴重（超過 50%），可以不畫。
*   **Q: 藥罐（圓柱體）標記難點？**
    *   A: 因為圓柱體在不同角度轉身後，標籤會被隱藏。建議標記至少 4-8 個不同旋轉角度的圖片，確保繞一圈都能被偵測到。

## 5. Roboflow 推介設定
*   **Dataset Versioning**：不要在原本的資料集上亂改，每次匯出前產出一個新版本 (e.g. `v1`, `v2`)。
*   **Preprocessing**：
    *   Auto-Orient: On
    *   Resize: Stretch to 640x640 (需與 YOLO `imgsz` 參數一致)
*   **Augmentation**：
    *   Horizontal Flip (左右翻轉)
    *   Brightness (亮度調整 +/- 25%)
    *   Blur (模糊處理，模擬相機微晃)
